博客搭建之路

发表于 2020-06-20 更新于 2020-06-21 阅读次数：

记叙了本博客搭建的经历，本博客的搭建大多是遵循网络上的教程，经过反复对比，一点点摸索出来的,（总而言之就是本人没有任何知识，弱鸡一个）,所以本文主要列举了给我帮助的文章，并稍作补充，以下正文——

  1. 知乎——从零开始搭建个人博客（超详细）
    我的环境配置完全依靠本文操作的，作者教的非常详细，并且在能在操作过程中感受到实现的逻辑，是非常好的一篇文章，同时也给了许多后期优化主题的方案，但本文在如何发博客方面介绍的并不清楚，详见下一篇文章；
  2. 知乎——hexo超完整的搭建教程，让你拥有一个专属个人博客
    本文在步骤方面稍显简略，（也可能是我太渣了）但我在本文中学会了发博客，原文这样说——

    接下来你就可以正式开始写文章了。
    hexo new newpapername
    然后在source/_post中打开markdown文件，就可以开始编辑了。当你写完的时候，再
    hexo clean
    hexo g
    hexo d

同时，本文也有一些优化方法，很有启发性；
3. CSDN——Hexo 添加分类及标签
通过本文学会了添加标签和分类；
4. CSDN——hexo 搜索功能
通过本文学会了添加搜索功能。
5. Hexo Next主题添加访客统计、访问次数统计、文章阅读次数统计
通过本文学会了添加访客、访问次数、文章阅读次数统计。
